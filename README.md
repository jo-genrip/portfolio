## 🛠️ 使用技術 (Tech Stack)

サイト自体の構築には、以下の技術・ライブラリを使用しています。

- **Frontend**: HTML5 / CSS3 / JavaScript (Vanilla JS)
- **Design Framework**: [Tailwind CSS](https://tailwindcss.com/) (視認性の高いUI設計)
- **Deployment**: GitHub Pages
